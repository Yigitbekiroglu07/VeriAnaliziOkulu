# 📊 Data Analysis School

This repository contains Jupyter Notebook materials designed for learning **Python programming and data analysis fundamentals**.

The project starts with **basic Python concepts** and gradually introduces essential data science tools such as **NumPy, Pandas, and data manipulation techniques**.

Each notebook includes both **conceptual explanations and practical exercises** to help learners practice and reinforce their knowledge.

---

# 📂 Contents

The repository includes the following Jupyter Notebook files:

## 1️⃣ Introduction to Programming

Covers the basic concepts of Python programming:

- What is Python
- Variables
- Lists
- Strings
- Dictionaries
- Operators
- Conditional statements (if)
- Loops
- Functions

---

## 2️⃣ List Exercises

Practice problems related to Python lists:

- Calculating the sum of elements in a list
- Finding the maximum and minimum values
- Creating and manipulating lists

---

## 3️⃣ Loops Exercises

Exercises related to loops and control structures:

- Counting even and odd numbers
- Generating the Fibonacci sequence
- Solving list problems using loops

---

## 4️⃣ Dictionary Exercises

Exercises for working with Python dictionaries:

- Accessing dictionary elements
- Modifying keys and values
- Adding new keys
- Removing keys

---

## 5️⃣ Functions Exercises

Practice writing Python functions:

- Function to find the largest of three numbers
- Calculating the product of numbers in a list
- Finding unique elements
- Prime number checking

---

## 6️⃣ NumPy Exercises

Numerical computing exercises using NumPy:

- Creating NumPy arrays
- Filtering array data
- Array reshaping
- Stacking arrays
- Calculating mean, median, and standard deviation
- Generating random numbers

---

## 7️⃣ NumPy & Matplotlib

This notebook covers:

- NumPy data structures
- Multi-dimensional arrays
- Vectorized operations
- Numerical computations
- Data visualization using Matplotlib

---

## 8️⃣ Data Manipulation with Pandas

Data analysis using the Pandas library:

- Series and DataFrame structures
- Data indexing
- Data filtering
- Mathematical operations
- Handling missing data
- MultiIndex usage
- Data merging (concat, merge, join)

# 🚀 How to Use

Clone the repository:

```bash
git clone https://github.com/yourusername/VeriAnaliziOkulu.git
```

Navigate to the project directory:

```bash
cd VeriAnaliziOkulu
```

Start Jupyter Notebook:

```bash
jupyter notebook
```

Open the notebooks and follow them in order.
